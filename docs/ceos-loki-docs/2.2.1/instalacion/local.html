Instalar y ejecutar Loki localmente

Para registrar eventos con Loki, debe descargar e instalar Promtail y Loki.

  • Loki es el motor de registro.

  • Promtail envía registros a Loki.

Instalar y ejecutar

  1. Navega a la página de lanzamiento.

  2. Desplácese hacia abajo hasta la sección Activos debajo de la versión que desea instalar.

  3. Descargue los archivos .zip de Loki y Promtail que correspondan a su sistema. Nota: No descargue LogCLI o Loki Canary en este momento. LogCLI le permite ejecutar consultas de Loki en una interfaz de línea de comandos. Loki Canary es una herramienta para auditar el desempeño de Loki.

  4. Descomprima el contenido del paquete en el mismo directorio. Aquí es donde se ejecutarán los dos programas.

  5. En la línea de comandos, cambie el directorio (cd en la mayoría de los sistemas) al directorio con Loki y Promtail. Copie y pegue los siguientes comandos en su línea de comandos para descargar archivos de configuración genéricos:

    wget https://raw.githubusercontent.com/grafana/loki/master/cmd/loki/loki-local-config.yaml
wget https://raw.githubusercontent.com/grafana/loki/main/clients/cmd/promtail/promtail-local-config.yaml

  6. Ingrese el siguiente comando para iniciar Loki:

    Windows

    .\loki-windows-amd64.exe --config.file=loki-local-config.yaml

    Linux

    ./loki-linux-amd64 -config.file=loki-local-config.yaml

Loki ejecuta y muestra los registros de Loki en su línea de comando y en http://localhost:3100/metrics.

¡Felicitaciones, Loki está instalado y funcionando! A continuación, es posible que desee editar el archivo de configuración de Promtail para introducir registros en Loki.

Paquetes comunitarios de openSUSE Linux

La comunidad proporciona paquetes de Loki para openSUSE Linux. Para instalar:

  1. Agregue el repositorio https://download.opensuse.org/repositories/security:/logging/ a su sistema. Por ejemplo, si está utilizando Leap 15.1, ejecute sudo zypper ar https://download.opensuse.org/repositories/security:/logging/openSUSE_Leap_15.1/security:logging.repo; sudo zypper ref

  2. Instale el paquete Loki con zypper in loki

  3. Habilite los servicios de Loki y Promtail:

    • systemd start loki && systemd enable loki

    • systemd start promtail && systemd enable promtail

  4. Modifique los archivos de configuración según sea necesario: /etc/loki/promtail.yaml y /etc/loki/loki.yaml.
