Instalar Loki

Métodos de instalación

Instrucciones para diferentes métodos de instalación de Loki y Promtail.

  • Instalar desde la fuente

  • Instalar y ejecutar localmente

Proceso general

Para ejecutar Loki, debe:

  1. Descargue e instale tanto Loki como Promtail.

  2. Descargue los archivos de configuración para ambos programas.

  3. Inicie Loki.

  4. Actualice el archivo de configuración de Promtail para obtener sus registros en Loki.

  5. Inicie Promtail.
